## Known Boundaries

- Local demo uses SQLite and development auth settings.
- RAG semantic vector search requires PostgreSQL pgvector and embedding configuration; local demo uses lexical fallback by default.
- Microsoft Teams external posting requires real Azure/Teams configuration; local demo covers tab, callback, summary, and queue behavior.
- AI provider calls are optional for the demo; heuristic fallback is the local reliability path.
